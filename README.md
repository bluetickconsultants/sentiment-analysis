<h3> IMDB Movie Review Sentiment Analysis </h3><hr>

This project is about movive reviews sentiment analysis based on Machine Learning , NLP and LSTM model.

<img src="/IMDB Movie Reviews Sentiment Analysis/Images/img1.jpg" width="700" height="400">

<h3> Using LSTM_RNN  model </h3><hr>

For developing sentiment Analysis model using LSTM Layers few techniques were applied for making the model to perform in better way 

<h3> Techniques used </h3>

     1 . Collecting Data from various sources
     2 . Text_cleaning
     3 . Balancing Data
     4 . Regular Expression
     5 . pipeline of NLP
            a . lower text
            b . stemming 
            c . lemmatization
            d . stopwords
            e . spacy library 
            f . nltk(natural language toolkit)
            
           
           
<h3> Accuracy result using word Embedding and LSTM </h3> <hr>
     
     1 . Training_Accuracy = 52.8942115768463
     2 . Test_Accuracy = 0.4898785425101215
